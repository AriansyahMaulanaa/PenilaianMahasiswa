# Aplikasi Penilaian Mahasiswa

Aplikasi Penilaian Mahasiswa merupakan aplikasi berbasis Android yang digunakan untuk menghitung nilai akhir mahasiswa secara otomatis berdasarkan beberapa komponen penilaian.

Aplikasi ini dirancang untuk memudahkan proses perhitungan nilai agar lebih cepat, akurat, dan mengurangi kesalahan perhitungan manual.

---

## Fitur Utama

Aplikasi ini memiliki beberapa fitur utama, yaitu:

* Input nilai UTS (Ujian Tengah Semester)
* Input nilai UAS (Ujian Akhir Semester)
* Input nilai Tugas
* Input nilai Kehadiran
* Perhitungan nilai akhir secara otomatis
* Penentuan grade (A, B, C, D, E)
* Tampilan hasil nilai yang sederhana dan mudah dipahami

---

## Cara Kerja Aplikasi

1. User memasukkan nilai UTS, UAS, Tugas, dan Kehadiran
2. Sistem menghitung nilai akhir berdasarkan bobot yang telah ditentukan
3. Sistem menentukan grade berdasarkan nilai akhir
4. Hasil akhir ditampilkan ke layar

---

## Teknologi yang Digunakan

Aplikasi ini dikembangkan menggunakan teknologi berikut:

* Java (Android Native Development)
* Android Studio (IDE)
* XML (User Interface)

---

## Tujuan Project

Tujuan dari pembuatan aplikasi ini adalah:

* Membantu proses perhitungan nilai mahasiswa secara otomatis
* Mengurangi kesalahan dalam perhitungan manual
* Sebagai media pembelajaran dalam pengembangan aplikasi Android
* Sebagai tugas atau project perkuliahan

---

## Catatan

Project ini masih dalam tahap pengembangan dan dapat dikembangkan lebih lanjut dengan penambahan fitur seperti penyimpanan data, histori nilai, atau integrasi database.
